## user Stories 

- As a user I can register and login.
- Simple Note app which is able to:
- Create, edit and delete notes.
- View a list of notes.
- Save notes in CoreData.

​
## Framework
- UIKit
- Firebase
- AVFoundation
- Core Data Saving support
